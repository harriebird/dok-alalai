# dok-alalai

An AI assistant for medical professionals.

## Requirements
- [Docker](https://www.docker.com/)
- [Ollama](https://ollama.com/)
- [Python](https://www.python.org/)
- [uv](https://docs.astral.sh/uv/)

## Setup

### Development

#### Using Docker:

1. Install [Docker](https://docs.docker.com/engine/install/) or [Docker Desktop](https://docs.docker.com/desktop/).
2. Clone this repository.
3. Install [Ollama](https://ollama.com/download).
4. Run `docker compose -f docker-compose.dev.yml up`
5. Go to `http://localhost:8000/` and enjoy!
